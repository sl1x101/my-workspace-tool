# Simple Workspace Launcher
A lightweight Python automation script designed for anyone who wants to start their workday with a single click. (Perfect for those who embrace "productive laziness"!)

## Features
- **Auto-Web Opener**: Launches your essential websites (Google, Pomofocus, Spotify) in separate tabs.
- **Sequential Loading**: Built-in 1-second delay between each tab to prevent system lag.
- **Local App Support**: Automatically opens your favorite local applications (like Notepad or VS Code).

##  How to Use
1. Make sure you have **Python** installed.
2. Clone this repository or download workspace.py.
3. Run the script: 'python workspace.py'
